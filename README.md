# accounting-for-finances
Приложение для учёта финансов в транспортной компании.

Описание приложения
Приложение имеет 4 формы. 
Первая форма authorization. В ней пользователь только авторизуется.
Вторая форма BoxOffice. Здесь идет основная работа. Эта  форма имеет 4 вкладки: проведение, история операций, касса и водители. Во вкладке "проведение" выбирается водитель, дата операции и все приходы/расходы выбранного водителя. В "Истории операций" соответственно можно увидеть историю всех операций и отсортировать её по предоставленным критериям. В "кассе" можно увидеть текущую сумму операций и изъять либо внести средства, которые не учитываются, но отображаются в истории (по сообраниям заказчика). В последней вкладке "водители" можно либо добавить нового сотрудника, либо уволить его.
Третья форма EditRow. Появляец при редактировании строки через вкладку "история операций".
Четвертая форма AreYouSure. Она предназначена для подтверждения удаления строки.

Под капотом касса работает так:
Σ(Наличные) + Σ(Автовокзал) - Σ(Топливо) - Σ(Запчасти)
