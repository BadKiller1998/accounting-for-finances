# accounting-for-finances
Приложение для учёта финансов в транспортной компании.

Описание приложения

Приложение имеет 4 формы.

Первая форма — Authorization. В ней пользователь только авторизуется.

Вторая форма — BoxOffice. Здесь идёт основная работа. Эта форма имеет 4 вкладки: "Проведение", "История операций", "Касса" и "Водители". Во вкладке "Проведение" выбирается водитель, дата операции и все приходы/расходы выбранного водителя. В "Истории операций" соответственно можно увидеть историю всех операций и отсортировать её по предоставленным критериям. В "Кассе" можно увидеть текущую сумму операций и изъять либо внести средства, которые не учитываются, но отображаются в истории (по соображениям заказчика). В последней вкладке "Водители" можно либо добавить нового сотрудника, либо уволить его.

Третья форма — EditRow. Появляется при редактировании строки через вкладку "История операций".

Четвёртая форма — AreYouSure. Она предназначена для подтверждения удаления строки.

Под капотом касса работает так:

Σ(Наличные) + Σ(Автовокзал) - Σ(Топливо) - Σ(Запчасти)

Для того чтобы приложение заработало, необходимо создать БД со структурой dump.sql. В TablesWithProfit\TablesWithProfit\WorkWithDB\QueryDB.cs нужно задать значение констант для работы с вашей БД. После этого в БД создать пользователя (в БД пароль хранится в зашифрованном варианте). Для шифрования пароля используйте функцию из TablesWithProfit\TablesWithProfit\HashPassword\HashPassword.cs.
